# Металл Монтаж 33

Лендинг компании по производству и монтажу металлических гаражей и навесов. Статический сайт без сборки и зависимостей — открывается в браузере как есть.

## Возможности

- Адаптивная вёрстка для телефона, планшета и десктопа
- Фиксированная шапка на мобильных с корректной прокруткой к разделам
- Карусели гаражей и навесов
- Блоки: типы кровли, услуги, дополнительные услуги, контакты
- Переключение тёмной и светлой темы
- Автовыбор темы по настройкам устройства (`prefers-color-scheme`)
- Сохранение выбранной темы в `localStorage`

## Стек

- HTML5
- CSS3 (CSS-переменные, Grid, Flexbox)
- Vanilla JavaScript (без фреймворков и npm)

## Запуск локально

Сайт не требует установки пакетов. Достаточно открыть `index.html` в браузере.

Через локальный сервер (удобнее для разработки):

```bash
# Python 3
python3 -m http.server 8080

# или npx (если установлен Node.js)
npx serve .
```

После запуска откройте в браузере: [http://localhost:8080](http://localhost:8080)

## Редактирование контента

Весь контент сайта собран в объекте `company` в файле `main.js`. Чтобы изменить текст, контакты или списки — правьте этот объект.

### Контакты и общая информация

```js
const company = {
  name: 'Металл Монтаж 33',
  phones: [
    { label: 'Алексей', value: '+7 (904) 254-36-74', href: 'tel:+79042543674' },
    { label: 'Евгений', value: '+7 (920) 343-47-27', href: 'tel:+79203434727' },
  ],
  email: 'MetallMontage33@yandex.ru',
  hours: 'Ежедневно 8:00–18:00',
  // ...
};
```

### Гаражи и навесы

Массивы `garages` и `canopies` управляют слайдерами. Каждый элемент содержит заголовок, описание и ссылку на изображение:

```js
garages: [
  {
    title: 'Гараж 6×4 м',
    size: '6000 × 4000 × 3600 мм',
    meta: 'Длина 6 м · Ширина 4 м · Высота 3,6 м',
    img: './assets/garage-6x4.jpg', // путь к вашему фото
  },
  // ...
],
```

Чтобы заменить плейсхолдеры на реальные фото — положите изображения в папку `assets/` и укажите путь в поле `img`.

### Услуги, кровля, доп. услуги

| Поле в `main.js` | Что отображает на сайте          |
| ---------------- | -------------------------------- |
| `services`       | Список основных услуг            |
| `roofs`          | Карточки типов кровли            |
| `extras`         | Дополнительные услуги            |
| `messengers`     | Ссылки на мессенджеры            |
| `hero`           | Заголовок и текст первого экрана |

## Тема оформления

- По умолчанию используется системная тема устройства
- Кнопка ☀ / 🌙 в шапке переключает тему вручную
- Выбор сохраняется в `localStorage` под ключом `mm33-theme`

Цвета задаются через CSS-переменные в `styles.css`:

| Переменная  | Назначение             |
| ----------- | ---------------------- |
| `--primary` | Оранжевый акцент       |
| `--bg`      | Фон страницы           |
| `--text`    | Основной цвет текста   |
| `--surface` | Фон карточек и панелей |

## Публикация

Сайт состоит только из статических файлов. Для публикации загрузите содержимое папки на любой хостинг:

- GitHub Pages
- Netlify / Vercel
- Обычный веб-хостинг (FTP)

Достаточно, чтобы `index.html` был доступен по корневому URL домена.

## SEO

Сайт уже содержит базовую SEO-оптимизацию:

- Title, description и keywords с региональными ключевыми словами
- Open Graph и Twitter Card для соцсетей и мессенджеров
- JSON-LD разметка (`LocalBusiness`, `WebSite`, каталог гаражей)
- `robots.txt` и `sitemap.xml`
- Гео-мета-теги для Владимирской области
- Семантическая разметка: один `h1`, иерархия `h2`/`h3`, alt у изображений

### Перед публикацией

1. Домен уже указан в `main.js`:

```js
seo: {
  siteUrl: 'https://quisty1.github.io/garages',
  serviceArea: {
    region: 'Владимирская область',
    cities: ['Владимир', 'Ковров', 'Муром', /* ... */],
  },
}
```

2. Зарегистрируйте сайт в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) и [Google Search Console](https://search.google.com/search-console), указав URL: `https://quisty1.github.io/garages/`

3. Замените плейсхолдеры фото на реальные — это улучшит поведенческие факторы и CTR в поиске.

### SEO-настройки в main.js

| Поле                     | Назначение                     |
| ------------------------ | ------------------------------ |
| `seo.siteUrl`            | Канонический домен сайта       |
| `seo.serviceArea.region` | Регион обслуживания            |
| `seo.serviceArea.cities` | Города в регионе               |
| `seo.ogImage`            | Картинка для превью в соцсетях |
