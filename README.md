# Лендинг «Гаражи под ключ» (моковые данные)

Файл-основа: `index.html`

Стили: `styles.css`  
JS (моковые данные + карусель/FAQ/форма): `main.js`

## Что менять, когда будут реальные данные

1. Название/контакты/адреса/мессенджеры — в `main.js` в объекте `company` (поле `messengers`: Telegram, WhatsApp, MAX).
2. Логотип — замените `assets/logo-placeholder.svg` на ваш файл (или обновите `src` у `<img>` в `index.html`).
3. Иконка вкладки браузера — замените `assets/favicon.svg` (подключена в `<head>` в `index.html`).
4. Фотографии для слайдеров — замените поля `img` в `company.portfolio` на ваши URL/пути.

Сайт статический: можно открыть `index.html` в браузере.
